This page demonstrates how to embed Dxsh dashboards and widgets using embed tokens.

üìä Dashboard Embed

Instructions:
  1. Go to your Dxsh dashboard at http://localhost:3001/embed-tokens
  2. Create a new embed token for a dashboard
  3. Copy the token and paste it below
  4. Enter your dashboard ID and click "Update Embed"

üéØ Widget Embed

Instructions:
  1. Create a new embed token for a widget
  2. Copy the token and paste it below
  3. Enter your widget ID and click "Update Embed"

üíª Embed Code Examples

Once you have working embeds above, you can use these HTML snippets in your own websites:

Dashboard Embed Code:

<iframe 
  src="http://localhost:8001/api/v1/embed/dashboard/YOUR_DASHBOARD_ID?token=YOUR_TOKEN&theme=light&refresh=30" 
  width="100%" 
  height="500"
  frameborder="0">
</iframe>

Widget Embed Code:

<iframe 
  src="http://localhost:8001/api/v1/embed/widget/YOUR_WIDGET_ID?token=YOUR_TOKEN&theme=light&refresh=30" 
  width="100%" 
  height="300"
  frameborder="0">
</iframe>
